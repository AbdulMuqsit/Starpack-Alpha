# Starpack Titan 0.1.7 to 0.1.8
### Important
- Backup your client settings and control keys before upgrading to this version
- Ore gen work is still underway, major update will come soon
### Bug Fixes
- Fixed crash when carry-on-picking plants
- Fixed Ambient sounds client crash
- Fixed terraforged reporting crash
- Fixed all entities going invisible when plants grow
- Fixed latest silent gear world gen crash
- Fixed fps stutter when harvesting trees
- Fixed some necessary missing starting itemsmcjtyl
- Fixed refined-storage server-startup crash/freeze
- Fixed crash with the latest version of CC:Tweaked DirectionUtil Class
- Fixed Mekanism cardboard boxes causing item duplications

### JEK Fixes
- Implemented separate drop stack key from drop item
- Fixed modifiers not working in certain containers (e.g. jei)
- Fixed forge conflict checks
- Updated the search algorithm, results should be more relevant and better sorted
- Implemented JEKKeyMapping api for multi-modifier keys
- RU translation, thanks to [DrHesperus](https://github.com/DrHesperus?tab=repositories)
- Added/Fixed Compatibility for mods:
  1. Don't Drop It
  2. Interactic
  3. ItemPhysic
- Fixed modifier only vanilla keys resetting on restart
- Fixed forge keys with default modifiers resetting on restart
- Updated keybinding to let keys with no modifier pass through
  when a modifier key is pressed, but no keymapping with the modifier
  is triggered.
